How should arrows scale with current speed?

Currently all arrows are 30×30px regardless of velocity. Here are some approaches to magnitude-based sizing.

A

Size + opacity scaling, hide below threshold

Arrow size scales from ~10px to ~36px based on velocity. Opacity fades too (faster currents are bolder). Below ~0.1 kn, hide the arrow entirely — it's basically slack.

B

Size scaling only, hide below threshold

Scale arrow size with velocity but keep all arrows fully opaque. Simpler visually — color already encodes speed, so opacity might be redundant. Hide below ~0.1 kn.

C

Replace near-slack arrows with dots

Below ~0.3 kn, swap the arrow for a small neutral dot (slack indicator). Above that, arrows scale with velocity. This preserves station locations on the map even at slack.
